# Global Airbnb Performance Dashboard

# Project Overview
This project is an interactive **Power BI Dashboard** built using Airbnb listings and reviews data.  
The dashboard was created as part of a guided learning project to understand how real-world datasets can be analyzed and represented visually using business intelligence tools.

The project focuses on exploring Airbnb performance across different cities, room types, hosts, and customer reviews through interactive charts, KPIs, and DAX calculations.

---

# Objectives
- Analyze Airbnb listing and review trends
- Understand market share across cities
- Compare room types and pricing
- Evaluate host verification and profile patterns
- Build cumulative review frequency analysis
- Learn dashboard design and data storytelling using Power BI

---

# 📊 Dashboard Features

## 1️⃣ Overview Dashboard
### Key KPIs
- Total Listings
- Total Cities
- Total Hosts
- Property Types
- Total Reviews

### Visualizations Included
- Listing growth trends over the years
- Room type comparison:
  - Entire Place
  - Private Room
  - Shared Room
  - Hotel Room
- Airbnb lifecycle analysis:
  - Introduction
  - Growth
  - Maturity
  - Decline
  - Reinvention
  - COVID-19 Impact

---

## 2️⃣ Market Share & Ratings Dashboard

### Analysis Included
- Market share by city
- Superhost vs Non-Superhost listings
- Cumulative percentage review frequency
- Average room prices by room type
- City-wise ratings comparison

### Rating Metrics
- Accuracy
- Cleanliness
- Communication
- Location
- Value
